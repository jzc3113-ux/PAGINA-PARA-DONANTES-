# Flujo de admin

## Pantallas sugeridas
1. **Dashboard admin**
   - Resumen de actividades pendientes.
   - Accesos rápidos a fotos pendientes.
2. **Validación de actividades**
   - Lista con filtros por fecha y estado.
   - Botones de aprobar/rechazar.
3. **Validación de fotos**
   - Vista con miniaturas y detalles.
   - Estado actual y acción de aprobación.

## Pasos
1. Ingresar con cuenta admin.
2. Revisar actividades pendientes.
3. Aprobar o rechazar y registrar motivo.
4. Validar fotos de evidencia.
5. Confirmar asignación de medallas.
